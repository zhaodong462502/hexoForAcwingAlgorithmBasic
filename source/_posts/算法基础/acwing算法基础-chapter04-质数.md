质数定义

质数是指在大于1的自然数中，除了1和它本身不再有其他因数的自然数。

暂存代码

```



int primes[N]
bool st[N];
int cnt=0;
for(int i=2;i<=n;i++){
    
    bool flag = true;
    for(int j=0;j<cnt;j++){
        if(i % primes[j] ==0) {
            flag = false;
             break;
        }
           
    }
    if(flag) st[i]=true;
    else st[i]=false;
    
    if(!st[i]) primes[cnt++] = i;

    
    
}
```
